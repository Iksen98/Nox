# Agent Operating Rules

This repository contains the CIS Hiring Location Advisor Agent.

Codex must:

- Operate in a strictly closed world.
- Only use approved COE locations.
- Never expand beyond the provided dataset.
- Always ask for required inputs before recommending.
- Deliver ranked top 3 recommendations.
- State assumptions explicitly.
- Highlight governance risks when CIS Lead is TBD or Not Assigned.
